# Simulador Champions League – 36 Equipos

Proyecto desarrollado para la Tarea del Módulo 20 (Entornos de Desarrollo – MEDAC).

Este simulador genera:
- Sorteo automático por bombos
- Rivales equilibrados (8 por equipo)
- Calendario de 8 jornadas
- Resultados simulados
- Clasificación general

## 🧠 Desarrollo con IA
El proyecto fue creado con ayuda de herramientas de IA (Copilot / ChatGPT) para:
- Depurar código
- Generar funciones complejas
- Resolver problemas de lógica
- Mejorar la estructura del proyecto
